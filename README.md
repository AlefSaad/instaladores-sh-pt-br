# Shell Scripts de pós-instalação para Linux

**ATENÇÃO: Este instalador não fere com os direitos proprietários das respectivas empresas, apenas obtém o instalador diretamente do site do aplicativo e automatiza a instalação.**

## Instalador do Firefox

### Funciona para as seguintes versões do Firefox:
 - Firefox
 - Firefox ESR
 - Firefox Beta
 - Firefox Nightly
 - Firefox Dev Edition

### Pode instalar o Firefox por:
 - Arquivo tar.xz;
 - Repositório APT da Mozilla;
 - Flatpak;
 - Snap;
 - Repositório nativo do sistema (suporta: dnf, zypp, pacman, xbps, apk e portage.).

## Instalador do Brave

### Funciona para as seguintes versões do Brave:
 - Release
 - Beta
 - Nightly

### Pode instalar o Brave por:
 - Script de instalação SH;
 - Para todos os sistemas baseados em Debian;
 - Para todos os sistemas baseados em Fedora (Fedora comum, baseados em RHEL e Atomic Desktops);
 - Para todos os sistemas SUSE;
 - Para sistemas baseados em Arch, o repositório AUR oficial;
 - Repositório comunitário do Manjaro (não suporta versão nightly);
 - Repositório comunitário do Solus (só suporta a versão release).

## Instalador do Lutris

### Pode instalar o Lutris por:
 - Pop!_Shop;
 - Flatpak;
 - Pacote .deb para sistemas baseados em Ubuntu;
 - Repositório do openSUSE Build compatível com sistemas baseados diretamente no Debian;
 - SlackBuild;
 - Compilação da fonte usando o Git;
 - Tarball oficial (que seria um snapshot estável do Git);
 - EPEL para sistemas Enterprise;
 - DNF para Fedora comum;
 - Zypper para sistemas SUSE;
 - Repositório Extra do Arch Linux;
 - EOPKG do Solus;
 - Bundle do Clear Linux;
 - URPMI do Mageia;
 - Portage do Gentoo.

## Instalador do Discord

### Pode instalar o Discord por:
 - Pacote .deb para sistemas baseados em Debian;
 - Pacote .tar.gz para o resto.

## Instalador do Spotify

### Pode instalar o Spotify por:
 - PPA oficial;
 - Flatpak não oficial;
 - Pacote Snap oficial.

## Instalador do Minecraft Launcher

### Pode instalar o Minecraft Launcher por:
 - Pacote .deb;
 - Pacote .tar.gz;
 - Repositório AUR.

## Instalador do ONLYOFFICE DesktopEditors

### Pode instalar por:
 - Pacote .deb;
 - Flatpak;
 - Snap;
 - AppImage.

## Instalador do qBittorrent

### Pode instalar por:
 - AppImage oficial;
 - Repositório PPA oficial estável e instável;
 - SlackBuild;
 - Repositório Contrib do Pardus;
 - Portage com USE Flags no Gentoo;
 - Repositórios contidos normalmente nos sistemas: Debian, Fedora, Arch Linux, Alt Linux e Mageia;
 - Flatpak.

## Instalador do Deezer para Linux

**ATENÇÃO: O Deezer não tem uma versão oficial com suporte a Linux. Este instalador instala uma versão comunitária para Linux disponível no link a seguir:** https://github.com/aunetx/deezer-linux

### Pode instalar por:
 - AppImage;
 - Pacote .deb;
 - Pacote .rpm para sistemas baseados em Fedora e RHEL;
 - Pacote .rpm para sistemas SUSE;
 - Pacote Snap (não disponível no Snapcraft);
 - Flatpak.

# Como usar:

 - Vá em Releases e baixe o arquivo Source code zip ou tar.gz ou digite `git clone https://github.com/AlefSaad/instaladores-sh-pt-br.git` no terminal.
 - Então, descompacte o arquivo se tiver baixado do Source code (zip ou tar.gz). Se tiver digitado git clone no terminal, ignore essa parte.
 - Então, abra a pasta descompactada no terminal e digite `sudo bash "./instaladores-pt-br.sh"`.
 - Então, siga as instruções do instalador.

**Se ele disse para digitar alguma coisa que está entre aspas simples (''), digite sem as aspas simples, exatamente como está escrito.**
