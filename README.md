# Shell Scripts de pós-instalação v1.0

**ATENÇÃO: Este instalador não fere com os direitos proprietários das respectivas empresas, apenas obtém o instalador diretamente do site do aplicativo e automatiza a instalação.**

## Instalador do Firefox

### Funciona para as seguintes versões do Firefox:
 - Firefox
 - Firefox ESR
 - Firefox Beta
 - Firefox Nightly
 - Firefox Dev Edition

### Pode instalar o Firefox por:
 - Arquivo tar.xz;
 - Repositório APT da Mozilla;
 - Flatpak;
 - Snap;
 - Repositório nativo do sistema (suporta: dnf, zypp, pacman, xbps, apk e portage.)

## Instalador do Brave

### Funciona para as seguintes versões do Brave:
 - Release
 - Beta
 - Nightly

### Pode instalar o Brave por:
 - Script de instalação SH;
 - Para todos os sistemas baseados em Debian;
 - Para todos os sistemas baseados em Fedora (Fedora comum, baseados em RHEL e Atomic Desktops);
 - Para todos os sistemas SUSE;
 - Para sistemas baseados em Arch, o repositório AUR oficial;
 - Repositório comunitário do Manjaro (não suporta versão nightly)
 - Repositório comunitário do Solus (só suporta a versão release)

## Instalador do Lutris

### Pode instalar o Lutris por:
 - Pop!_Shop
 - Flatpak
 - Pacote .deb para sistemas baseados em Ubuntu
 - Repositório do openSUSE Build compatível com sistemas baseados diretamente no Debian
 - SlackBuild
 - Compilação da fonte usando o Git
 - Tarball oficial (que seria um snapshot estável do Git)
 - EPEL para sistemas Enterprise
 - DNF para Fedora comum
 - Zypper para sistemas SUSE
 - Repositório Extra do Arch Linux
 - EOPKG do Solus
 - Bundle do Clear Linux
 - URPMI do Mageia
 - Portage do Gentoo
 
## Instalador do Discord

### Pode instalar o Discord por:
 - Pacote .deb para sistemas baseados em Debian
 - Pacote .tar.gz para o resto

## Instalador do Spotify

## Instalador do qBittorrent

### Pode instalar por:
 - AppImage oficial
 - Repositório PPA oficial estável e instável
 - SlackBuild
 - Repositório Contrib do Pardus
 - Portage com USE Flags no Gentoo
